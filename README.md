# ScotiabankCapstone2019

## Start CMS
1. Open a Terminal
2. type "cd cms"
3. type "strapi develop"

Default local port is 1337.

username: admin

pass: admin123

This will run the Strapi CMS in dev mode

## Start Website
1. Open another Terminal
2. type "cd website"
3. type "npm start"

Default local port is 3000.

This will Start the Website
